# Changelog

All notable changes to VoiceHub are documented in this file. This project follows semantic versioning once public releases are published.

## [0.1.0] - 2026-06-23

### Added

- macOS Tauri desktop app for refining Chinese spoken development requests into Codex-ready prompts.
- Doubao Seed-ASR integration, configurable LLM refinement, local diagnostics, global triggers, prompt history, and clipboard handoff.
- Refined/raw output switching, Markdown preview, editing, copy feedback, and optional automatic copy.
- Public repository documentation, contribution guidance, security policy, issue templates, CI, and dependency update configuration.
